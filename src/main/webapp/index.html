Welcome to the Pirate Supply Store API!







Public endpoints:
Home: GET / Locations: GET /stores Login Page: GET /login Signup Page: GET /register
Pirate endpoints:
Submit registration form: POST /register Submit login form form: POST /register View past orders: /orders
Employee endpoints:
See inventory for a location: GET /stores/{location}/inventory All pirate clients: GET /pirates All orders: GET /orders Replenish inventory for a location: POST /stores/{location}/inventory
